HTML

    Every HTML document must start a document type declaration
  • The head element contains information about the webpage.
  • the head is the element that contains the metadata for our web application
  • the metadata is the data about our webpage that the browser uses to effectively serve our webpage
  • the head contains also the title the element that defines the title for our webpage which is required and is extremely important for Search Engine Optimization (SEO).
  • The body element represents the visible content shown to the user.
  • the headings are defined with h1 to h6 tags h1 is the most important one

CSS

  • CSS is the language we use to style a Web page
  • A CSS rule consists of a selector and a declaration block
  • The selector points to the HTML element you want to style
  • Each declaration includes a CSS property name and a value, separated by a colon
  • Each declaration includes a CSS property name and a value, separated by a colon

  • The CSS margin properties are used to create space around elements, outside of any defined borders
  • The CSS border properties allow you to specify the style, width, and color of an element's border
  • The CSS padding properties are used to generate space around an element's content, inside of any defined borders
